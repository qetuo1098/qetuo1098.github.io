Marginalizing and Conditioning Gaussians onto Linear Approximations of Smooth Manifolds with Applications in Robotics

Zi Cong Guo, James R. Forbes, Timothy D. Barfoot

International Conference on Robotics and Automation (ICRA), 2025

★ ICRA Best Conference Paper Award

Develops a framework for conditioning and marginalizing Gaussian distributions onto linearized manifolds, with connections to covariance extraction in constrained estimation.

Provides practical tools for extracting and manipulating uncertainty in constrained robotics estimation problems, especially when states live on geometric spaces.

Robots need to reason about uncertainty. This work helps them represent uncertainty correctly when working with quantities like position, orientation, and maps.

KILO-EKF: Koopman-Inspired Learned Observations Extended Kalman Filter

Zi Cong Guo, James R. Forbes, Timothy D. Barfoot

Submitted to International Conference on Intelligent Robots and Systems (IROS), 2026

Learns lifted observation functions and incorporates them into an EKF on SE₂(3), enabling approximately linear-Gaussian measurement updates with learned models.

Integrates learned observation models into a standard Extended Kalman Filter for robot localization, preserving familiar filtering structure while improving sensor modeling.

Helps robots figure out where they are by learning how their sensors behave from data, instead of relying only on hand-written mathematical models.

Koopman Linearization for Data-Driven Batch State Estimation of Control-Affine Systems

Zi Cong Guo, Vassili Korotkine, James R. Forbes, Timothy D. Barfoot

Robotics and Automation Letters (RA-L), 2022

Extends Koopman-inspired lifting to both process and measurement models, enabling approximately linear-Gaussian batch estimation for nonlinear control-affine systems.

Replaces difficult-to-model system dynamics with learned linearized representations, allowing standard batch estimation techniques to be applied to complex nonlinear systems.

Helps robots estimate hidden quantities such as position and velocity by learning how their motion and sensors behave directly from data.

Data-Driven Batch Localization and SLAM Using Koopman Linearization

Zi Cong Guo, Frederike Dümbgen, James R. Forbes, Timothy D. Barfoot

Transactions on Robotics (T-RO), 2024

Introduces Reduced Constrained Koopman Linearization (RCKL) for localization and SLAM, enforcing consistency between lifted and original state representations during batch optimization.

Enables data-driven localization and SLAM while preserving physically meaningful robot trajectories and uncertainty estimates, improving robustness compared to unconstrained approaches.

Helps robots build maps and determine their location using learned models while ensuring that the resulting maps remain physically consistent.

Smoothing Out the Edges: Continuous-Time Estimation with Gaussian Process Motion Priors on Factor Graphs

Connor Holmes, Sven Lilge, Zi Cong Guo, Frank Dellaert, Timothy D. Barfoot

Submitted to Foundations and Trends in Robotics, 2026

Provides a factor-graph interpretation of Gaussian-process continuous-time state estimation, demonstrating that trajectory interpolation and querying can be understood through standard elimination operations. This unifies Gaussian-process trajectory estimation with the factor-graph framework used throughout localization and SLAM, and provides practical implementations within GTSAM.

Many localization and SLAM systems must combine asynchronous measurements while estimating the robot trajectory between sensor times. Common examples include rolling-shutter cameras and sensors operating at different rates. Modeling the trajectory as a continuous-time Gaussian process helps resolve these timing mismatches by representing motion as a smooth path that can be queried at any time. The paper shows how to implement this approach within standard factor-graph pipelines in the widely used GTSAM library, with practical interfaces and examples for adoption in real-world robotics systems.

Robots often receive information from cameras, lasers, and other sensors operating at different times. In these situations, we can improve navigation by estimating the robot's motion as a smooth path rather than a sequence of disconnected snapshots. This work demonstrates that these methods can be understood and implemented using the same efficient estimation tools already used in many robotics systems.
